Three Bad Doors

多步任务里，最危险的是三扇门

每一扇门背后，都是 Agent 的“好心办坏事”。

发散之门从当前任务，跑到未来流程。
幻觉之门从模糊要求，长出新字段。
自检之门从完成声明，伪装成验收。
The Turn

别让 Agent 当出题人。
让它成为解谜玩家。

玩家只能拿当前线索。通过验证，才能看下一张。

Clue Cards

Step Contract 就是线索卡

每张卡只描述当前关卡，不泄露整张地图。

CLUE 01读取输入，只做 fetch
CLUE 02判断字段，只填格子
CLUE 03验证通过，解锁报告

Six Fields

一张卡，必须写清六件事

输入从哪里拿，不猜。
任务只做什么，不扩展。
输出产物在哪，不含糊。
验证怎么过关，不自检。
失败处理失败怎么办，不乱试。
下一步通过去哪，不推理。
Bound The Magic

LLM 可以判断，但必须进格子

自由文本越多，幻觉空间越大。字段越明确，行为越可控。

Agent may fill:
relevant: yes / no / uncertain
category: bug / feature / question / other
severity: high / medium / low
summary: 80 字以内中文

Agent must not rewrite:
raw content / url / author / date
Validation Gate

真正的门，不听解释

Agent 说“我检查过了”不算。exit code 才算。

python -c "
assert d['total'] == len(d['items'])
assert item['category'] in allowed
assert item['summary'].strip()
print('PASS')
"

PASS unlocks the next clue.
What Changes

ARG 不是让模型永远正确。
它是让错误更少发生、更早暴露、更容易拦截。
