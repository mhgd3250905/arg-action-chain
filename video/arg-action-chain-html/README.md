# ARG 行动链路动态 HTML 演示

这是一个知识分享类动态 HTML，适合全屏播放或录屏成视频。

## 打开方式

直接用浏览器打开：

```text
index.html
```

或使用自动播放模式：

```text
index.html?autoplay=1
```

## 控制

- `← / →`：上一页 / 下一页
- `Space`：播放 / 暂停
- `N`：显示 / 隐藏演讲稿
- `R`：录屏模式，隐藏控制条
- `F`：全屏

## 建议录屏流程

1. 打开 `index.html?autoplay=1`
2. 按 `F` 全屏
3. 按 `R` 进入录屏模式
4. 使用 OBS、系统录屏或浏览器录制工具录制
